Problema a Resolver: Facilitar la administración de una biblioteca, permitiendo a los usuarios consultar información sobre libros, autores y miembros de la biblioteca.
Clases:

    Libro: atributos como título, autor, género y número de páginas; métodos para ver la información del libro y actualizar el estado (disponible, prestado).
    Autor: atributos como nombre, nacionalidad, lista de libros escritos; métodos para agregar libros o ver la lista de publicaciones.
    Miembro: atributos como nombre, número de identificación, libros en préstamo; métodos para registrar préstamos y devoluciones de libros.

Acciones del Usuario: Consultar libros, buscar información de un autor, registrar y devolver libros.
